B == Design
  
  This program has two parts.
  
  The first part is a singly linked list called SingleLinkedList<T>.
  It keeps track of:
  	head (first node)
  	tail (last node)
  	size (number of elements)
  
  Each node stores a value and a reference to the next node.
  
  The list allows adding, removing, inserting, and finding items.
  
  If the list is empty, some methods return null.
  
  
  The second part is a stack called IntStack.
  It uses an ArrayList to store values.
  
  The top of the stack is the end of the list.
  
  It supports:
  	push
  	pop
  	peek
  	size
  	isEmpty
  	average
  
  If the stack is empty, some methods throw an error.
  
  The program uses a menu so the user can test all functions by typing input.
